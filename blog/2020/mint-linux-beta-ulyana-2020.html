Linux Mint is great operating system. It is based on the excellent Ubuntu and features three great desktop environment options. While it is a smart choice for Linux beginners, it is also good for experts too.

As all of the Linux Mint 19 code names are based on the letter 'T', the next set of releases will use code names based on the letter 'U'.
The Linux Mint 20 codename is 'Ulyana'.

Major Updates

  • Linux 5.4 kernel
  • Fractional scaling support
  • New file transfer app
  • More theme colours
  • Faster file manager
  • Change monitor refresh rate
  • Better support for multi-monitors

Linux Mint 20 is a long-term support release until 2025. However, in a break with previous releases, Linux Mint 20 is 64-bit only.
